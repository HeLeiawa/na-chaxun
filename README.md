# 钠云查询系统 (CX-NA)

一个基于PHP的品牌信息查询与认证系统，提供品牌查询、注册、证书生成、管理后台等功能。

## ✨ 功能特性

- 🔍 **品牌查询**：通过识别码、品牌名称、官网等关键词快速检索品牌信息
- 📝 **品牌注册/投稿**：用户可提交品牌信息，自动生成唯一识别码
- 🏅 **证书生成**：为合规品牌生成官方认证证书（支持电子版和实物证书）
- 🛡️ **状态管理**：品牌状态分类（经营、歇业、停业、跑路）
- 📊 **数据统计**：网站访问量、品牌数量等数据统计
- 👥 **友情链接**：支持添加和管理友情链接
- 📢 **公告系统**：后台发布和管理网站公告
- 🛒 **订单管理**：实物证书订单处理（待发货、已发货、无法发货）
- 🔐 **后台管理**：管理员登录，全功能后台管理系统

## 🚀 快速开始

### 环境要求
- PHP 5.6+（推荐PHP 7.0+）
- 支持JSON文件读写
- 支持文件上传（可选）
- 现代浏览器支持

### 安装步骤
1. 将项目文件上传到支持PHP的Web服务器
2. 确保以下文件有写权限（如果需要上传功能）：
   - `brands.json`
   - `notices.json`
   - `friends.json`
   - `stats.json`
   - `orders.json`
3. 访问网站首页，开始使用

### 默认后台账号
- 用户名：`admin`
- 密码：`197350`

**安全提示：** 首次使用后请立即修改后台密码！

## 📖 使用指南

### 品牌查询
1. 在首页搜索框输入品牌名称、识别码或关键词
2. 系统将显示匹配的品牌信息
3. 可查看品牌状态、官网、联系方式等详细信息

### 品牌注册
1. 点击"注册/投稿"按钮
2. 填写品牌名称、官网、介绍、状态等信息
3. 系统自动生成唯一识别码
4. 提交后品牌信息将添加到系统中

### 证书生成
1. 在证书页面输入品牌识别码
2. 系统验证品牌状态（仅"经营"和"歇业"状态可生成证书）
3. 生成电子证书，可下载或打印
4. 如需实物证书，可提交申请并支付

### 后台管理
1. 访问 `admin.php` 并登录
2. 管理功能包括：
   - 添加/编辑/删除品牌
   - 发布/管理公告
   - 管理友情链接
   - 查看统计数据
   - 处理实物证书订单
   - 更新订单发货状态
   - 删除不需要的订单
